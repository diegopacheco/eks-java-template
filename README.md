# eks-java-template

eks-java-template: Java 8 + Jetty 9 + Guice Docker + Kubernetes + EKS + Gradle + Rest Template

## Build with Docker
```bash
docker build -t diegopacheco/javajetty . --no-cache
```

## Run with Docker
```bash
docker run diegopacheco/javajetty
```

## Deploy Docker hub
```bash
docker push diegopacheco/javajetty
```
